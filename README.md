# React-Hooks-Crud

# Simple application developed using:

```sh
- functional component
- arrow functions
- useContext
- useReducer
```

## How to install packages (frontend)

### after cloning the repository use yarn or npm to run the application

### yarn

```sh
$ yarn
```

### npm

```sh
$ npm install
```

## How to run application

### yarn

```sh
$ yarn start
```

### npm

```sh
$ npm start
```

### Listening port on 3000

```sh
$ http://localhost:3000/
```
